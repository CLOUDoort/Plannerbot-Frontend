# Plannerbot
Planner Bot은 여행 장소와 일자를 기반으로 여행 계획을 제공하는 ChatGPT 기반 AI 여행 플래너 서비스이다.
맞춤형 추천 장소와 활동을 일 별로 제공하여 여행 계획을 세우는 과정을 간소화할 수 있다.
